CogniFit

Cognitive impairment is when a person has trouble remembering, learning new things, concentrating, or making decisions that affect their everyday life. Cognitive impairment ranges from mild to severe. CogniFit is a website which aims to help people with cognitive decline and help them to prevent it with amazing features.

Features

CogniFit helps you to prevent cognitive decline with amazing features.
The following features are:

Game.

We have designed a puzzle game as it helps develop coordination, fine motor skills and visual skills. It can help improve your concentration as well.

To-Do List.

Using a To-Do list makes you a super productive and organized person. We included a notification feature where you will be able to get the work done on time.

Emotion Camera.

Using this you can able to identify the emotion of the person and then can help them according to their emotional needs .
